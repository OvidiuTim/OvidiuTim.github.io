2.smart school

This project was an opportunity to learn more about web development. It has a simplistic layout mostly done with HTML and CSS, Navbar with a drop-down menu, different reactive tables using JavaScript and ngx-bootstrap. The tables represented the teachers and students of a hypothetical school. Each student had a name, a phone number and a subject attributed to them, and each teacher had a subject. All this information was in an SQL server. Using Postman I made and web API so I could change details about each student and teacher. Everything was made offline with a node server.
