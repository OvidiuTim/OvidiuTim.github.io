Universo Potencial was a freelancer job I did while living in Brazil. The purpose of the company was to promote online educational courses. The website was developed to showcase their mission, their team and the online courses available. The page was created using Elementor on Wordpress.



In worldpress I've added CSS special effects for hover, also change and define the containers using HTML for the phone interface.

2.smart school

This project was an opportunity to learn more about web development. It has a simplistic layout mostly done with HTML and CSS, Navbar with a drop-down menu, different reactive tables using JavaScript and ngx-bootstrap. The tables represented the teachers and students of a hypothetical school. Each student had a name, a phone number and a subject attributed to them, and each teacher had a subject. All this information was in an SQL server. Using Postman I made and web API so I could change details about each student and teacher. Everything was made offline with a node server. Check the code source with the QRcode.

3.PIZZA DE CASA

While living in Brazil, I owned a Pizza place and desired to create a website to present our menu and take orders from our customers. Unfortunately, this never came to be. The draft of the site was developed with a friendly interface, it has an order button that takes the visitor directly to our Whatsapp. Scrolling down is the restaurant's menu, a slideshow of the pizzas available and a short about us section with buttons for our social media.
